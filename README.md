# FastAPI Learning Projects

This repository contains my FastAPI learning assignments.

## Day 1 - Learning Management System (LMS)
Course enrollment platform with Clean Architecture implementation.

**Features:**
- Student registration
- Course management
- Enrollment system with duplicate prevention
- REST APIs with Swagger documentation

**Run Day 1:**
```bash
cd day1-lms
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Visit: http://localhost:8000/docs
